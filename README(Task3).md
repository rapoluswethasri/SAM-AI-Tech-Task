# Task 3 – Votes Analysis

## Overview

This project analyzes restaurant voting patterns to understand customer engagement and restaurant popularity.

## Objective

* Identify restaurants with the highest and lowest number of votes.
* Analyze the relationship between customer votes and restaurant ratings.
* Discover insights about restaurant popularity and customer preferences.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab / VS Code

## Dataset

The dataset includes:

* Restaurant Names
* Aggregate Ratings
* Votes
* Cost Information
* Location Details

## Analysis Performed

### Highest Votes Analysis

* Identified restaurants receiving the maximum customer votes.
* These restaurants are considered highly popular among customers.

### Lowest Votes Analysis

* Identified restaurants with very few customer votes.
* Lower votes may indicate lower visibility or fewer customer interactions.

### Correlation Analysis

* Compared restaurant ratings with vote counts using visualization techniques.
* Used scatter plots and correlation methods to analyze trends.

## Visualizations Created

* Bar Chart – Restaurants with Highest Votes
* Bar Chart – Restaurants with Lowest Votes
* Scatter Plot – Votes vs Ratings
* Heatmap – Correlation Analysis

## Key Insights

* Restaurants with higher ratings generally receive more votes.
* Popular restaurants attract greater customer engagement.
* Some restaurants have good ratings but fewer votes due to lower exposure.
* Votes can be used as an indicator of restaurant popularity.
* A positive correlation exists between votes and ratings.

## Conclusion

The analysis demonstrates that customer votes and restaurant ratings are closely connected. Restaurants delivering better customer experiences tend to receive higher ratings and more engagement from customers.
