# Data Visualization Project

## Overview
This project focuses on transforming raw restaurant data into meaningful visual insights using Python data visualization libraries.

## Objective
To analyze restaurant ratings, votes, and costs through professional charts and graphs.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Dataset
The dataset contains:
- Restaurant Names
- Ratings
- Votes
- Cost Information

## Visualizations Created
1. Bar Chart – Restaurant Ratings Analysis
2. Pie Chart – Votes Distribution
3. Scatter Plot – Ratings vs Votes Analysis
4. Line Chart – Restaurant Cost Analysis

## Key Insights
- Restaurants with higher ratings received more customer votes.
- Restaurant D showed the highest popularity and ratings.
- Cost trends were clearly identified through line visualization.
- Data visualization simplifies understanding of complex data.

## Conclusion
Data visualization helps transform raw data into understandable graphical formats. Professional visualizations improve analysis, storytelling, and decision-making.
